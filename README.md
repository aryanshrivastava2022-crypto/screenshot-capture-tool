# Screenshot Capture Tool
A simple Python application that captures screenshots and saves them automatically.

# Features
- Capture full-screen screenshots
- Save screenshots with timestamp-based filenames
- Easy-to-use command-line interface
- Automatic image storage

# Requirements
- Python 3.x
- PyAutoGUI Library

# Installation
Install the required library:
pip install pyautogui

# How to Run
1. Download the project files.
2. Open a terminal in the project folder.
3. Run:
screenshot_capture.py
4. Press Enter when ready.
5. The screenshot will be saved automatically.
